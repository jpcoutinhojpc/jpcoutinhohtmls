# Gerador de GC

Ferramenta web estática para criar GC's de jornalismo, entretenimento e esporte com controle de animações, relógio, cores de fundo e compressão inteligente de texto.

## Como usar
1. Abra `index.html` em um navegador moderno.
2. Preencha título, subtítulo e tipo de GC.
3. Ajuste a largura máxima (px) para definir o limite onde o texto começa a se comprimir.
4. Escolha mostrar/ocultar o relógio e selecione a cor de fundo (verde, azul ou transparente para chroma key).
5. Selecione o estilo de animação e use os botões **Mostrar GC** ou **Ocultar GC** para visualizar as transições.

O texto é comprimido horizontalmente e tem a fonte reduzida quando necessário para permanecer dentro da largura configurada.
